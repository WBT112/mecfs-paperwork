# Help

## English (EN)

### What this app does

- **Formpacks** are curated sets of forms for a specific need.
- **Cases** are your saved work items for a formpack, stored offline in your browser.
- **Tools** guide you through filling, validating, and exporting documents.

### Export workflow (DOCX)

1. Open a case.
2. Review and complete all required fields.
3. Choose **Export** and select **DOCX**.
4. Save the file to your device.

### Open and edit DOCX files

**PC (Windows / macOS)**

- Recommended apps: Microsoft Word, LibreOffice, or OnlyOffice.
- Typical steps: open the app → open the exported file → edit → save.

**Android**

- Recommended apps: Microsoft Word (mobile), Collabora Office, or OnlyOffice.
- Typical steps: download the file → open with the app → edit → save locally.

**iOS / iPadOS**

- Recommended apps: Microsoft Word or Apple Pages.
- Typical steps: open the file from Files → edit → save or export.

### Privacy reminder

All data stays on your device. Exported files are stored where **you** choose.

## Deutsch (DE)

### Wie die App funktioniert

- **Formpacks** sind kuratierte Formulare für einen bestimmten Zweck.
- **Fälle** sind deine gespeicherten Arbeitsstände zu einem Formpack – offline im Browser.
- **Tools** helfen beim Ausfüllen, Prüfen und Exportieren.

### Export (DOCX)

1. Öffne einen Fall.
2. Prüfe und ergänze alle Pflichtfelder.
3. Wähle **Export** und anschließend **DOCX**.
4. Speichere die Datei auf deinem Gerät.

### DOCX-Dateien öffnen und bearbeiten

**PC (Windows / macOS)**

- Empfohlene Apps: Microsoft Word, LibreOffice oder OnlyOffice.
- Typische Schritte: App öffnen → Datei öffnen → bearbeiten → speichern.

**Android**

- Empfohlene Apps: Microsoft Word (mobil), Collabora Office oder OnlyOffice.
- Typische Schritte: Datei herunterladen → mit der App öffnen → bearbeiten → lokal speichern.

**iOS / iPadOS**

- Empfohlene Apps: Microsoft Word oder Apple Pages.
- Typische Schritte: Datei in „Dateien“ öffnen → bearbeiten → speichern oder exportieren.

### Datenschutz-Hinweis

Alle Daten bleiben auf deinem Gerät. Exportierte Dateien werden dort gespeichert, wo **du** es auswählst.
